# Init

This init system aims for simplicity and performance. It also provides a easy to understand config written in TOML that makes it very easy to make changes to the system.

## Features

- **Simplified Config:** This init offers a straightforward config for creating and modifying services and other options.
  
- **Multithreaded:** Services will be started in parallel to improve startup speeds.

- **Lightweight:** This init system is very lightweight and has a very small codebase.

- **IPC**: The init system will make use of IPC to be interacted with.

## Installation

This init system is currently only compatible with PoopyPooOS and has not been tested on other distributions. It is also very unlikely that it will work for other distributions.
